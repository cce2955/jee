This page calculates a proposed "Tribonacci Sequence", in a normal sequence you would use N = N + N1.
For example, a sequence starting at N=1, and N1 = 1, would create 1 + 1 = 2, in turn 2 + 1 = 3, 2 + 3 = 5, 5 + 3 =8, etc. etc.
A "Tribonacci" does the same with 3 values, 1 + 1 + 1 = 3, 1 + 1 + 3 = 5, 1 + 3 + 5 = 9, 3 + 5 + 9 = 17, etc.
